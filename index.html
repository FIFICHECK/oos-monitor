⚙️ How to Add / Remove SKUs

用 terminal command 加 SKU：

# Add a SKU
python3 ~/oos-monitor/manage_skus.py add B0961005_S_XXXXXXXXXXXX

# Add multiple at once
python3 ~/oos-monitor/manage_skus.py add B0961005_S_XXX
python3 ~/oos-monitor/manage_skus.py add B0961005_S_YYY

# List all monitored SKUs
python3 ~/oos-monitor/manage_skus.py list

# Remove a SKU
python3 ~/oos-monitor/manage_skus.py remove B0961005_S_XXXXXXXXXXXX

更新 config 後，下一個 cron job run 就會自動生效。
